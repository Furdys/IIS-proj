Implementace

Při implementaci jsme využili open-source PHP Framework Laravel. Na úrovni front front-endu byl využit framework bootstrap. Dále byl využit framework Javascriptu jQuery. Implementovany byli všechny případy užití ze zadaní IDS/IUS s případným přejmenováním některých případu (objednání na košík). ERD vyžadoval více změn - původný ERD neřešil případy, kdy se v jednu chvíli provádí operace nad různými druhy léku - například prodej prodej různych druhů léku (ne různeho počtu). Ze zadání IIS rovněž vyplýva potřeba přidat entitní množinu pro uživatele a role.

  • jQueryUi 1.12

Postup instalace

  1. Vytvoříme nový Laravel projekt pomocí composer create-project --prefer-dist laravel/laravel IIS-proj
  2. Do nově vytvořené složky IIS-proj překopírujeme soubory z archivu xfurda00.zip
  3. Inicializujeme databázi provedením příkazů v souboru database-dump.sql
  4. V souboru .env upravíme přístupy k MySQL databázi
  5. Pomocí příkazu php artisan serve spustíme server a web je poté dostupný na adrese localhost:8000
